# 安装依赖

先执行 `npm install` 安装依赖。然后有如下命令可用：

- `npm test`  运行单元测试
- `npm run js` 编译JS
- `npm run less` 编译less
- `npm run watch` 进入watch模式 自动编译文件

# 运行
- `npm server` 即可！
